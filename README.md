# Customer Type Exploration

This mini project focuses on exploratory data analysis and data quality checks
using a dataset that contains customer identifiers and customer type information.

## Scope
- Customer type distribution
- Percentage analysis
- Duplicate and consistency checks

## Dataset
- File: customers.csv
- Columns: CustomerID, customer_type
- - Format: CSV


## Limitation
The dataset contains only categorical variables.
Therefore, no numeric or Excel-based summary outputs were generated.
